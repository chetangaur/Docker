Docker Volumes:-
   Volumes are the preferred mechanism for persisting data 
   generated by and used by Docker containers.

Use of Volumes:-
 1. Decoupling container from storage
 2. Share volume(storage/data) among different containers
 3. Attache volume to container
 4. On deleting container volume does not delete

# How to store different containers data to same volume or how multipe containers use same volume as storage
Create a volume v1 that store data of container 'A'
 docker run --name NameOfContainer -v NameOfVolume:/var/jenkins_home -p 8080:8080 -p 50000:50000 ImageName
      -> -v to allocate volume
      -> data transfer or store from /var/Image_home to the volume you created
 
Store data of container 'B' to same volume v1       
 docker run --name NameOfContainer -v NameOfVolume:/var/Image_home -p 9090:8080 -p 60000:50000 ImageName

#  To store data in local host machine  
 docker run --name MyJenkins3 -v /Users/raghav/Desktop/Jenkins_Home:/var/jenkins_home -p 9191:8080 -p 40000:50000 jenkins

Basic commands related with docker volumes:-
 
1. docker volume  //get information
2. docker volume create
3. docker volume ls
4. docker volume inspect
5. docker volume rm
6. docker volume prune
